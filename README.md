📚 BookStore E-commerce - Web Application
¡Bienvenido a BookStore! Este es un proyecto de aplicación web interactiva, adaptativa y persistente desarrollado como parte del Proyecto de Octavo Semestre. Se trata de una tienda en línea de venta de libros donde los usuarios pueden explorar un catálogo, comprar libros y gestionar su tarjeta de membresía.

📖 Características principales
Catálogo de libros con imágenes almacenadas en Azure Blob Storage

Búsqueda y filtrado de libros por título, ISBN, género, autor y precio

Compra de libros con descuento del saldo de tarjeta de membresía

Recarga de saldo (mínimo $50.000, máximo $200.000)

Autenticación de usuarios por correo y tarjeta de membresía

Historial de compras para clientes

Panel de administración para gestionar inventario y usuarios

Registro de transacciones en Cosmos DB (incluye IP, fecha, hora)
